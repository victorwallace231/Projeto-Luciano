#Projeto_de_seleção_para_Luciano

##Landing_para_Luciano_Feijão

Projeto de páginas, para divulgação do novo semestre 2026.2 com o
fluxograma de objetivo levar o usuário a se interessar e inscrever-se
em algum dos cursos de oferecidos pela universidade 

###Fluxo
Passo 1: No index.html (primeira página), o usuário lê o conteúdo da página ou clica no botão "inscrever-se já"
ou clica no link da barra de navegação "cursos" para acessar os cursos disponiveis.
Passo 2: O mesmo usuário caso tenha interesse em alguma das ofertas clica em "Saiba Mais" e é levado a uma página
com um detalhamento do curso escolhido.
Passo 3: O usuario finalmente após escolher o curso, clica em increver-se para iniciar o processo de inscrição para
o novo semestre do ano.
